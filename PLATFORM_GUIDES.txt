TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════

Problem: "ModuleNotFoundError: No module named 'youtube_transcript_api'"
Solution: Run: pip install youtube-transcript-api

Problem: "FileNotFoundError: batch_X.csv"
Solution: Make sure you uploaded the CSV and the filename matches exactly

Problem: Getting "Blocked" status repeatedly
Solution: That platform is rate limited. Switch to a different platform
          and continue from where it stopped

Problem: Script finishes too fast (< 1 minute)
Solution: Check if INPUT_CSV path is correct and file has video IDs
